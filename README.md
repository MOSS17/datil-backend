# Datil API

Backend API for Datil — an appointment booking platform for small service businesses.

## Tech Stack

- **Go** with [chi](https://github.com/go-chi/chi) router
- **PostgreSQL** via [pgx/v5](https://github.com/jackc/pgx) (raw SQL, no ORM)
- **JWT** authentication ([golang-jwt](https://github.com/golang-jwt/jwt))
- **WhatsApp** notifications via Twilio (behind a swappable interface)
- **Docker** for local development

## Getting Started

### Prerequisites

- Go 1.25+
- Docker and Docker Compose
- [golang-migrate](https://github.com/golang-migrate/migrate) CLI

### Setup

```bash
# Start PostgreSQL
make docker-up

# Configure environment
cp .env.example .env

# Run migrations
make migrate-up

# Start the server
make run
```

The API will be available at `http://localhost:8080`.

## Project Structure

```
├── cmd/api/               # Application entry point
├── internal/
│   ├── config/            # Environment and database configuration
│   ├── handler/           # HTTP handlers (one file per domain)
│   ├── middleware/         # JWT auth, CORS
│   ├── model/             # Go structs matching DB tables
│   ├── repository/        # Database queries (one file per domain)
│   ├── router/            # chi router setup and route registration
│   └── notification/      # Notifier interface + Twilio implementation
├── migrations/            # SQL migration files (golang-migrate)
├── docker-compose.yml
├── Dockerfile
└── Makefile
```

| Method | Path | Description |
|---|---|---|
| POST | `/auth/signup` | Create business + user account |

| GET | `/book/{url}/availability?date=YYYY-MM-DD` | Available time slots |
| POST | `/book/{url}/reserve` | Create a reservation |

## Database

11 tables managed via golang-migrate. Migrations are in `migrations/` and ordered by foreign key dependencies. All tables use UUID primary keys generated by PostgreSQL (`gen_random_uuid()`).

To create a new migration:

```bash
make migrate-create name=add_some_column
```
